### 使用>

##### 查单词:  利用好say

```shell
tr hello
```

##### 时间戳准换

```shell
util formatTime 1630143508789
```

##### 压缩 svg

```shell
util svgo 路径 （默认为./svg）
```

##### 修改图片名字

```shell
util photo 路径 （默认为./photo）
```
