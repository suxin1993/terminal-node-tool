### 使用方法

下载以后打开 cmd，运行

`npm install`

安装所有依赖

* 运行简历词云爬虫

  `node index.js`

* 运行草榴爬虫，这是之前的代码，网址应该失效了，网址对的话代码应该可以跑通，我相信你们

  `node caoliu.js`

* 这是头条街拍的爬虫

  `node toutiao.js`
